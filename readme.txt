WP Version Check

== Description ==

This plugin allows you to manually check for new wordpress versions if 
you don't want to wait up to half a day for the cron tasks to run.

More info:

* [WP Version Check plugin](http://patchlog.com/wordpress/wp-version-check/).
* Check out the other [Wordpress plugins](http://patchlog.com/wordpress/) by the same author.


== Installation ==

The "WP Version Check" plugin can be installed by following this steps:

1. Unzip "wp-version-check.zip" archive and put all files into your "plugins"
folder (/wp-content/plugins/) or create a sub directory into the
plugins folder (recommanded), like /wp-content/plugins/wp-version-check/

2. Activate the plugin

3. go to Wordpress Admin -> Tools -> Version Check when you want to check for
new versions of wordpress
